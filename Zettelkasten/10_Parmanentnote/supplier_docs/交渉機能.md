# 交渉機能

## 📋 交渉プロセス管理

### 交渉フロー
1. **交渉開始**: 査定完了後の交渉開始
2. **交渉進行**: 売主との交渉・調整
3. **条件確定**: 価格・条件の合意
4. **承認申請**: 内部承認プロセス
5. **契約準備**: 契約書類の準備

### 交渉状態管理
- **交渉中**: 売主との交渉進行中
- **条件調整中**: 価格・条件の調整中
- **承認待ち**: 内部承認待ち
- **契約準備中**: 契約書類準備中
- **交渉完了**: 交渉プロセス完了
