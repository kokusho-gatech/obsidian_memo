# HealthCheckController

## 役割
このコントローラはアプリケーションのヘルスチェック（死活監視）用のエンドポイントを提供します。

## 処理フロー
```mermaid
graph TD
  "A"["リクエスト受信"] --> "B"{"アクション分岐"}
  "B" -->|"index"| "C"["ヘルスチェック応答"]
```

## アクション一覧
- `index`: ヘルスチェック用の応答を返す（通常は200 OKや簡易なテキスト）。

## コールバック
- 特になし（認証不要でアクセス可能な場合が多い）。

## Strong Parameters
- なし
